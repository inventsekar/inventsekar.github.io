Lets learn some vi editor search and replace: vi / vim editor - beginner to advanced - options - search and replace 1). Search whole file, replace globally (all occurrence/repetitions will be replaced) :%s/FindMe/ReplaceMe/g 2). Search current line only, replace globally on that line only. :s/FindMe/ReplaceMe/g 3). Search whole file, replace only first match :%s/FindMe/ReplaceMe/ 4). Search current line only, replace first match only. :s/FindMe/ReplaceMe/ 5). Search whole file, replace globally, with confirmation(c). :%s/FindMe/ReplaceMe/gc 6). Search whole file, replace globally, with confirmation(c), Case Sensitive(I) search. :%s/FindMe/ReplaceMe/gcI 7). Search whole file, replace globally, with confirmation(c), Case insensitive(i) search. :%s/FindMe/ReplaceMe/gci
